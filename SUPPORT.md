# Support

## Getting Help

- Usage questions and feature discussions: open a GitHub Issue.
- Bugs: open a GitHub Issue with repro steps and environment details.
- Security concerns: use the private channel in `SECURITY.md`.

## What to Include

- OS and architecture.
- Rust version (`rustc --version`).
- Command used and full error output.
- Minimal reproduction if possible.

## Maintainer Expectations

Best-effort support is provided. Response time is not guaranteed.
